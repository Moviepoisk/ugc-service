### Причины выбора MongoDB:
- Может быть использован в связке с самостоятельным сервисом, для последующей интеграции в etl процесс.
- Простота настройки и использования из коробки.
- Возможность шардирования и горизонтального масштабирования.
- Высокая скорость вставки данных и работа с большим объемом данных.
- Возможность менять схему данных без необходимости изменения самой базы данных.
